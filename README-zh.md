# 第2周实验仓库 — Ledger

这是一个小型的、可正常运行的订单/计费服务：创建订单、收款、退款。

它可以正确运行，代码也不凌乱——这是有意为之的，与第1周不同。问题不在于代码本身，而在于代码**周围**没有被记录下来的一切：没有 CLAUDE.md、一个自上线后就再未更新过的 `docs/` 目录，以及一个真实的产品决策（`CHANGELOG.md`）——只有仔细阅读的人才会在动手修改之前发现它。

在完成实验第1步之前，先不要去翻 `CHANGELOG.md`——你需要先亲身感受那个"空白"。详见 `LAB_GUIDE-zh.md`。

## 运行

    pip install -r requirements.txt
    pytest
